## Description

This is the backend repo for our project using NestJS.

You can visit our site here: [UniFood](https://github.com/) (In progress)

## Installation

```bash
$ npm install
```

## Running backend

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
